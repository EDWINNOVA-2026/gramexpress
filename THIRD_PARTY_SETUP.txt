GramExpress Third-Party Setup
=============================

1. Copy `.env.example` to `.env`

2. Google authentication
- Create a Google Cloud OAuth client for web
- Add your local/production origins
- Put the client ID in `GOOGLE_CLIENT_ID`
- Google sign-in currently connects existing accounts and redirects new users into registration
- If popup mode is unreliable on mobile/PWA, GramExpress uses the redirect-to-server GIS flow

3. Email OTP delivery
- Demo mode:
  `EMAIL_BACKEND=django.core.mail.backends.console.EmailBackend`
- SMTP mode:
  `EMAIL_BACKEND=django.core.mail.backends.smtp.EmailBackend`
  `EMAIL_HOST=smtp.gmail.com`
  `EMAIL_PORT=587`
  `EMAIL_HOST_USER=your-email@gmail.com`
  `EMAIL_HOST_PASSWORD=your-app-password`
  `EMAIL_USE_TLS=true`
  `EMAIL_TIMEOUT=20`
  `DEFAULT_FROM_EMAIL=GramExpress <your-email@gmail.com>`
  `SITE_URL=https://yourdomain.com`

4. Mobile OTP delivery
- Demo mode:
  `SMS_BACKEND=console`
- Twilio mode:
  `SMS_BACKEND=twilio`
  `SMS_FROM=+1...`
  `TWILIO_ACCOUNT_SID=...`
  `TWILIO_AUTH_TOKEN=...`

5. Maps
- `GOOGLE_MAPS_BROWSER_API_KEY`
- `GOOGLE_MAPS_EMBED_API_KEY`

6. Payments
- `RAZORPAY_KEY_ID`
- `RAZORPAY_KEY_SECRET`
- `RAZORPAY_WEBHOOK_SECRET`
- Full guide: `RAZORPAY_SETUP.md`

7. Local media
- Uploads are stored in `media/`
- Django serves them automatically while `DEBUG=true`
